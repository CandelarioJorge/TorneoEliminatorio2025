Este es un proyecto demostrativo sobre la creación de herramientas digitales para su implementación dentro del aula

Instrucciones:

•	Se debe de cargar un archivo con extensión .csv
•	El archivo debe de contener encabezados de columna con la siguiente notación:
o	Nombres
o	ApellidoPaterno
o	ApellidoMaterno
•	De no seguir estas instrucciones no se generará la página de manera correcta
